### File Handlers (Core)

* [CSV File Handler](): Extracts documents from a `CSV` file.
* [JSON File Handler](): Extracts documents from a `JSON` (or a `JSONL`) file.
* [XML File Handler](): Extracts documents from an `XML` file.
